## 二、理论框架升级

### 2.1 预测压缩原理 v2

```
原始信号 S (高维)
    ↓ 预测器 f
预测 S' = f(S_t)
    ↓ 压缩
误差 E = ||S - S'||
    ↓
如果 E < 阈值:
    表征 Z = encode(S) 形成
    ↓
Z 的意义: 最小描述长度(MDL)最优
```

### 2.2 涌现机制

| 机制 | 说明 |
|------|------|
| 竞争 | 表征空间有限 |
| 合作 | 预测器共享特征 |
| 分化 | 新表征分裂 |
| 稳定 | 能量奖励形成惯性 |

## 三、与现有理论的关系

### 3.1 对比预测处理(Friston)

| 方面 | Friston | AES |
|------|---------|-----|
| 核心 | 能量最小化 | 压缩涌现 |
| 表征 | 推断 | 涌现 |
| 学习 | 变分 | 演化 |

**AES贡献**: 预测压缩 → 表征涌现的完整路径

### 3.2 对比主动推理

主动推理: 感知 → 行动 → 反馈
AES: 预测 → 表征 → 行动 → 涌现

## 四、形式化定义

### 4.1 基本元素

- **环境**: E = (S, A, T)
  - S: 状态空间
  - A: 动作空间  
  - T: 转移函数

- **Agent**: A = (φ, θ, ω)
  - φ: 感知函数
  - θ: 预测网络
  - ω: 动作网络

- **资源约束**: R = (M, C)
  - M: 内存上限
  - C: 计算上限

### 4.2 涌现条件

```
涌现发生 当且仅当:
1. R 有限
2. T 非平稳
3. ∃ 策略 π: 预测误差 ↓
4. 信息可继承
```

## 五、关键洞见

### 5.1 表征的本质

> **表征不是"对世界的复现"，而是"对预测有用的压缩"**

### 5.2 对象的涌现

```
早期: 特征 → 像素模式
中期: 模式 → 稳定轨迹
后期: 轨迹 → 对象概念
```

### 5.3 因果的涌现

```
推动箱子 → 箱子移动 (因果链)
A推动 → B移动 → C到达 (因果链)
```
