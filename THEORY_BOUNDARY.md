### 1.2 边界条件

**定义1: 可涌现环境**
```
E ∈ U (可涌现环境) ⟺ 
  1. R(E) < ∞        (资源有限)
  2. T(E) is cyclic  (非平稳/循环)
  3. |A| ≥ 2         (至少2个动作)
  4. S ≥ 3           (至少3个状态)
```

**定义2: 涌现**
```
Emergence(E, A) ⟺ 
  ∃π: 策略, t: 时间
  使得: F(π_t) > F(π_0) × (1 + ε)
  其中 ε > 0 是涌现阈值
```
