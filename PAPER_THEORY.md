## 2. 边界定义

### 2.1 可涌现环境
```
E ∈ U ⟺ 
  1. R(E) < ∞
  2. |A| ≥ 2
  3. S ≥ 3
```

```
Emergence ⟺ ∃π: F(π_t) > F(π_0) × (1+ε)
```
