## 一、表征维度上界

### 问题
表征维度D的最大值是多少？

### 解决思路

基于**信息论**和**资源约束**：

```
设:
- R: 资源总量(内存/计算)
- S: 状态空间大小
- D: 表征维度

约束: D * log(S) ≤ R

上界: D ≤ R / log(S)
```

**经验估计**:
- 网格世界(10x10): S=100, log(S)≈4.6
- 实际资源R≈1000维度
- D ≤ 1000/4.6 ≈ 200

**实验观察**: 我们的实验中有效表征约10-20维

## 二、涌现时间尺度

### 问题
涌现需要多长时间？

### 解决思路

基于**品均理论**：

```
τ_涌现 = O(1/λ)

λ = 选择压力 × 变异率 × 继承效率
```

**实验测量**:
- 阶段1: 约100-200步涌现对象追踪
- 阶段2: 约50-100步涌现因果理解
- 阶段3: 约200-300步涌现策略

**公式**:
```
τ = τ_0 * (1 + 1/λ)
其中 τ_0 = 基础学习时间
```

## 三、通用性证明

### 问题
AES是否适用于其他环境？

### 解决思路

**必要条件充分性**：

```
AES适用 ⟺ 满足四个必要条件:
1. R有限 (资源约束)
2. 反馈闭环 (环境可响应)
3. 变异+选择 (演化机制)
4. 信息继承 (知识传递)

这些条件在大多数环境中满足
```

**特例**:
- 纯随机环境 → 不适用(无模式可预测)
- 完全确定性 → 可能不涌现(无需预测)
- 混沌边缘 → 最适合涌现
