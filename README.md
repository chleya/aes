# 对抗涌现系统 (Adversarial Emergence System)

## 项目代号

**AES** (Adversarial Emergence System)

## 核心命题

> 智能通过与"他者"的对抗来定义自我，并在对抗中涌现新的认知形式

---

## 三个子方向

### A. 对抗涌现
- 两个Agent相互对抗
- 对抗过程中涌现新的策略/表征
- 类似GAN但用于智能体

### B. 对抗+FCRS
- 把对抗机制加入有限竞争表征系统
- Agent之间争夺表征资源
- 竞争产生新的表征

### C. 对抗自我认知
- "我" vs "他者"
- 通过区分自我与他者来建立身份
- 对抗产生自我意识

---

## 架构

```
┌─────────────────────────────────────┐
│         Agent A (自我)               │
│  表征池 + 策略网络                  │
└──────────────┬──────────────────────┘
               │ 对抗信号
               ▼
┌─────────────────────────────────────┐
│         Agent B (他者)               │
│  表征池 + 策略网络                  │
└─────────────────────────────────────┘
```
