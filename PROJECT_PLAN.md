# AES 项目评价与规划书

**日期**: 2026-03-07

## 一、项目目标

验证核心命题：**表征系统能否从简单规则中涌现**

> 通过"预测驱动压缩"，验证有限资源系统能否自发形成可泛化的结构表征。

## 四、理论贡献

### 4.1 预测压缩原理

```
预测未来（压缩状态）
    ↓
发现持续稳定模式
    ↓
内部涌现：对象概念
    ↓
预测更准确 → 能量奖励
```

### 4.2 与现有理论的关系

| 理论 | 对应 |
|------|------|
| 预测处理(Friston) | 预测→能量 |
| MDL压缩 | 预测=压缩 |
| 演化计算 | 变异+选择 |
