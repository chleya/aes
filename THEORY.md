## 二、理论基础

### 2.1 必要条件（4条）

| # | 条件 | 说明 |
|---|------|------|
| 1 | 资源有限 | 否则无限记忆 |
| 2 | 反馈闭环 | 否则无法学习 |
| 3 | 变异+选择 | 否则无法演化 |
| 4 | 信息继承 | 否则无法积累 |

### 2.2 时间尺度分离

```
τ_learning << τ_evolution

快时间：神经活动、行为学习
慢时间：基因演化、结构优化
```

## 三、智能涌现层级

### 3.1 渐进路径

```
阶段1：B移动实体 → 对象追踪 → 对象概念
阶段2：C可交互实体 → 因果理解
阶段3：多Agent → 社会预测 → Theory of Mind
阶段4：自我预测 → 自我意识
```

### 3.2 四种智能形式（对应不同路线）

| 形式 | 核心 | 涌现什么 |
|------|------|----------|
| A | 物理结构 | 空间理解 |
| B | 因果结构 | 预测能力 |
| C | 语义结构 | 概念关系 |
| D | 目标结构 | 行动规划 |

**选择**：B → C渐进（以预测为核心）

## 四、核心机制

### 4.1 预测压缩原理

```
预测未来（压缩状态）
    ↓
发现持续稳定模式
    ↓
内部涌现：对象概念（latent）
    ↓
预测更准确 → 能量奖励
```

### 4.2 预测 vs 其他目标

| 目标 | 问题 |
|------|------|
| 预测像素 | 噪声主导，学不到结构 |
| 预测预定义对象 | 失去"涌现"意义 |
| 预测奖励 | 只是反射，无理解 |
| **预测压缩状态** | **让系统自己发现重要的** |

### 4.3 避免Collapse

三种办法：
1. 信息约束：限制latent维度
2. 对比学习：区分正确vs随机
3. 行为耦合：预测好→更多能量

## 五、实验设计

### 5.1 最小系统

```
环境：32x32网格
  - 移动实体（食物/捕食者）
  - 统计规律（周期、轨迹）

Agent：
  - 感知：3x3局部视野
  - 网络：input→hidden→output（权重可遗传）
  - 动作：移动/停留

资源：
  - 能量 -= 1/步
  - 食物 += 20
  - 危险 = 死亡

竞争：
  - 能量超阈值 → 复制
  - 复制时变异：高斯噪声
```

### 5.2 预测奖励机制

```
正确预测(t+Δ) → 获得能量
错误预测 → 损失能量

Δ = 3或5（必须预测未来）
```

### 5.3 观测指标

| 指标 | 测量 |
|------|------|
| 行为复杂度 | trajectory entropy |
| 内部结构分化 | PCA/聚类权重 |
| 预测能力 | 预测准确率 |
| 信息压缩 | I(agent_state; future_env) |

### 5.4 成功信号

1. Agent内部状态出现稳定聚类
2. 不同群体对应不同策略
3. 涌现非预设的行为模式
