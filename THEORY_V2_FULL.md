## 一、核心框架

### 1.1 预测压缩原理 (正式定义:
- S_t)

设: t时刻状态
- f: 预测函数
- E: 能量函数

```
预测: S'_t = f(S_{t-1})
误差: ε = ||S_t - S'_t||
能量: E = -log(P(S_t|S'_{t-1}))
```

**涌现条件**:
当 ε < 阈值时，形成表征 Z = encode(S)

### 1.2 必要条件定理

```
涌现发生 ⟺ (R有限) ∧ (T非平稳) ∧ (∃π: 预测↓) ∧ (可继承)
```
