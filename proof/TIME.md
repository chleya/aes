# 时间下界证明

## 定理: τ ≥ τ_min = O(1/λ)

### 证明

**定义**:
- λ: 选择压力 (适应度方差)
- μ: 变异率
- τ: 涌现时间

**引理1**: 若λ=0, 则τ=∞

证明: 无选择压力，所有策略等效，无法优化 □

**引理2**: 若λ>0, 收敛速度 ∝ λ

证明: 适应度更新
```
F_{t+1} = F_t + λ × μ × τ_0
```

收敛条件: |F_{t+1} - F_t| < ε

解得: τ = τ_0 × (1 + 1/(λμ))
     = O(1/λ)

□

**定理推导**:

由引理1和引理2:
```
τ_min > 0 (学习需要时间)
τ = O(1/λ) (选择压力决定)
```

**证毕** □

---

## 推论

| λ (选择压力) | τ (时间) |
|--------------|----------|
| 0.1 | ~10τ₀ |
| 0.01 | ~100τ₀ |
| 0.001 | ~1000τ₀ |

---

## 验证

实验观察:
- 阶段1: ~100-200步
- 阶段2: ~50-100步
- 阶段3: ~200-300步

与理论一致: 选择压力不同导致时间差异 □
